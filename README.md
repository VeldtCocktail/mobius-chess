# Möbius Chess
A chess variant, inspired by this here [video](https://www.instagram.com/reel/DZpjyD0Dmfj/?utm_source=ig_web_copy_link&igsh=MzRlODBiNWFlZA==)

## Set up a virtual environment in Python
```python -m venv {environment-directory}``` <br>
⚠️ Remember to add your virtual environment directory to your personal .gitignore, as it will be pushed to the directory if not.
