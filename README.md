# Möbius Chess

A chess variant, inspired by this here [video](https://www.instagram.com/reel/DZpjyD0Dmfj/?utm_source=ig_web_copy_link&igsh=MzRlODBiNWFlZA==)

The game follows the basic rules of chess, with the specificity that any piece can _phase_ through the board if the square on the _other side_ is unoccupied. The "other side" square refers to the square that has the coords (_row + 8 / row - 8_, _opposite column_)

## Set up a virtual environment in Python

`python -m venv {environment-directory}`
⚠️ Remember to add your virtual environment directory to your personal .gitignore, as it will be pushed to the directory if not.
